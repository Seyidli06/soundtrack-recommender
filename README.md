# 🎵 Soundtrack Recommender

A Java application that recommends movie soundtracks based on the user's mood using the Spotify Web API.

## Features
- Mood-based soundtrack recommendations
- Spotify API integration
- Rule-based recommendation algorithm

## Tech Stack
- Java
- Maven
- Spotify Web API
- Gson
